 I'm an experienced web designer with a passion for creating fun and interesting games and tools. I'm mostly self taught besides some beginner javascript classes. I've been doing some form of coding since 2020. I started in scratch for a year, then I started learning HTML and CSS. I started programing in Javascript in 2024, mainly in p5.js, a easy to use javascript library specialized for interactive projects.


 I was inspired to have a site by YouTube creators like Mark Rober and Veritasium's passion for teaching in creative ways. I also just really liked videogames. My first browser game was an incremental game called Corn Country. It was really just a Cookie Clicker knockoff with Corn Syrup and Popcorn. I actually learned how to do this from a YouTube tutorial titled How to make an incremental/clicker game by Tanktot Games. This video still only has 40k views (2025) but it lead to me learning more Javascript and making this entire website in 2023.


Speaking of this site, it's open source! You can view the repostiory on Github here. My site has been hosted on a couple of different sites. This page was hosted on Neocities for a year and a half, on 000webhost for a couple months (until 000webhost closed down in July 2024), and has been hosted on Github Pages since September 29th, 2024.
